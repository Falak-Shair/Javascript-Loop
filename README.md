# Javascript-Loop

This contain following tasks :

Q1): A 5-digit positive integer is entered through the keyboard, write a function to calculate sum of digits of the 5-digit number.

Q2) Write a program to find the factorial value of any number entered through the keyboard.

Q3) Write a program to find the range of a set of numbers. Range is the difference between the smallest and biggest number in the list.

Q4) Write a program to print all prime numbers from 1 to 300.

 Q5) Write a program to print the multiplication table of the number entered by the user. The table should get displayed in the following form.  
        29 * 1 = 29      
        29 * 2 = 58
